poker: end-of-hand win/payout cheer on the shared board. When a hand resolves (and at tournament end) the board overlays a celebration — won chips fly from the pot to each winner, confetti bursts, a "+amount" badge pops, the winning seat plate glows gold, and a centered banner names the winner with the made hand (or "uncontested" on a fold-around; a larger gold "wins the tournament" finale at game over). Split pots / side pots show one badge per winner — the engine emits a payout per pot layer, so the overlay sums per seat. Chip amounts (the pot and the won piles) are drawn with a new `ChipPile` whose chip count is proportional to the actual amount, broken into denominations. Because the overlay is a pure function of state it renders in both replays and live play; replay playback now dwells on the settle frame so the cheer is watchable, via an optional per-frame hold added to `usePlayback`. New API: `payoutMs` + `resultHoldMs` props on `PokerBoard`, the `ChipPile` export, and the `LIVE_PAYOUT_MS` (3s gameplay) / `REPLAY_PAYOUT_MS` (1s replay) cheer-duration constants. Depends on the poker engine now emitting each hand's resolution as its own `hand_complete` tick (carrying `pot_distribution` / `showdown_hands`); previously that state was overwritten by the next deal within the same `step` and never reached replays.
